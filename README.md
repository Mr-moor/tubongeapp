# TubongeApp

**TubongeApp** is a Python-powered social media platform designed to provide users with a modern and engaging social networking experience. The app combines content feeds, verified accounts, groups, tags, stories, and video features into a single platform.

---

## Features

### User Accounts & Profiles
- Secure registration and login with **Python-based authentication**
- **Verified accounts** for authenticity
- Customizable profiles with **privacy and account settings**
- Support for **groups, tags, and search**

### Social Interaction
- Users can post **videos, stories, and text content**
- Like, comment, and share posts
- **Follow friends** and interact in a community
- Direct messaging between users

### Content & Discovery
- Algorithm-driven **feeds** for personalized content recommendations
- Discover trending posts, hashtags, and videos
- Stories and videos created in-app or uploaded

### Notifications & Settings
- Notifications for **mentions, messages, likes, and updates**
- Full **account and privacy management** in the settings area
- Community guidelines to maintain a safe environment

---

## Technologies Used

- **Backend**: Python (Flask/Django)  
- **Database**: SQLite or other Python-supported databases  
- **Frontend**: Optional Flutter mobile interface  
- **Security**: Verified accounts, password hashing  
- **Video & Stories**: Python libraries for handling media content  
- **Algorithm**: Feed personalization combining features similar to TikTok, Facebook, Twitter, and YouTube
